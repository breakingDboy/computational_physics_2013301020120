>###[第六次作业](https://www.zybuluo.com/Gwater/note/340589)
>实现了在考虑风阻，但忽略科里奥利力和地球形状的情况下，精确计算炮弹发射速度打击目标的功能

>结果分析:

>Origin :x = 0 , y = 0 
Traget: x = 10000  , y=1000
vx=vy=277.166
error_range = 0.08
relative_error=8*10^(-6)
![结果](http://i.imgur.com/L2vjUJ2.png)
可见这样的算法省时而又非常精确的找出了炮弹的初始发射速度
